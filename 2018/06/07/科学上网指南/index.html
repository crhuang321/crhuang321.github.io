科学上网指南

为了科学上网和进一步学习，我决定动手搭建一个自己的服务器。在搭建过程中遇到了一些阻碍，这里非常感谢大神的帮助 @PlatypusPro 。本文记录搭建服务器、使用 Digital Ocean 和 Shadowsocks 实现科学上网的过程。

1 注册DigitalOcean账号

首先注册一个属于你自己的Digital Ocean账号，此时你需要绑定信用卡💳，或者使用你的PayPal账户支付5$，方可使用。这里附上我的邀请链接 http://www.digitalocean.com/?refcode=804fc47b7367 ，使用此邀请链接注册你的Digital Ocean账户，你将获得10$。

另外，如果你是在校学生，使用你的 .edu 邮箱通过 GitHub Education 验证，你的Digital Ocean账户可再得50$。

2 创建Droplet云主机

你可以根据你的实际情况来选择。选择前记得对各节点主机测速，比如 San Francisco 1 。我选择的配置为：1 GB Memory / 25 GB Disk / SFO2 - Ubuntu 16.04.4 x64 。
